grabd
====
grabd - 数据收集接口汇总

# 初衷
用于定向爬取和采集idol数据，并分析来源的走向。（当然是为了小偶像了!）
	
	
# 介绍
分析定向爬虫接口，可当做爬虫的匹配规则使用，各类涉及到搜索的接口。	

目前暂支持:	
* 百度	
* 微博	
* 58同城	
* 知乎	
	
	
# 需要用到的外部库
* bs4.BeautifulSoup	
* requests	
* MySQLdb	
	
	
# 目录结构
* pet_58data	
	* data58.py - 58同城宠物数据	
* data_48	
	* baidu_data.py - 百度搜索接口	
	* zhihu_data.py - 知乎搜索接口	
	* weibo_data.py - 知乎搜索接口
